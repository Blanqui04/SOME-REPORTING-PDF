## [5.0.0] - 2025-01-01

### 🚀 Features

- *(backend)* FastAPI application with JWT auth, LDAP, TOTP 2FA
- *(backend)* PDF generation with WeasyPrint, compression, AES-256 encryption
- *(backend)* Celery task queue for async report generation
- *(backend)* Multi-template system with overlay and watermark support
- *(backend)* Grafana dashboard/panel rendering via HTTP API
- *(backend)* Report scheduling with cron expressions
- *(backend)* Audit logging for security events
- *(backend)* Multi-tenant organization support
- *(backend)* Grafana alerting webhook for auto-report on alert
- *(frontend)* React SPA with Vite, Tailwind CSS, dark mode
- *(frontend)* Dashboard browser with search and panel selection
- *(frontend)* Report management with download and deletion
- *(frontend)* Drag & drop panel reordering (@dnd-kit)
- *(frontend)* Side-by-side report comparison
- *(frontend)* PWA support with offline caching
- *(frontend)* Multi-language support (CA, ES, EN, PL)
- *(docker)* Full Docker Compose setup (app, PostgreSQL, Redis, Celery)
- *(helm)* Kubernetes Helm chart with HPA, Ingress, Secrets
- *(tests)* 230+ unit/integration tests with pytest
- *(tests)* E2E tests with Playwright
- *(tests)* Load testing with Locust
